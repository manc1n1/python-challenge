## Installation

1.  Clone the repo

    ```sh
    git clone https://github.com/manc1n1/python-challenge.git
    ```

2.  Change directories to `python-challenge`

    ```sh
    cd python-challenge
    ```

3.  Open in `VSCode` or `Spyder` and run the code

    ```sh
    python main.py
    ```
